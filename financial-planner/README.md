# financial-planner

> 基于 [Nuxt.js](https://github.com/nuxt/nuxt.js) (vue) 构建的项目

## Build Setup

``` bash
# install dependencies
初始化项目, 安装依赖包:
$ npm install # Or yarn install

# serve with hot reload at localhost:3010
开发模式
$ npm run dev

# build for production and launch server
正式环境的打包
$ npm run build
$ npm start

# generate static project
生成静态前端文件
$ npm run generate
```
